# Images Directory

Place your images here:

1. **logo.png** (or logo.svg) - Your DreamForge logo
   - Recommended size: 120x80px or similar aspect ratio
   - Will be displayed in a rectangle above the title

2. **background.jpg** (or background.png) - Cosmic nebula background
   - Recommended: High resolution (1920x1080 or larger)
   - Will be used as the full-page background

## Current Setup

- Logo path: `/images/logo.png`
- Background path: `/images/background.jpg`

Update the paths in `app/auth/[[...rest]]/page.tsx` and `app/auth/sign-in/page.tsx` if you use different filenames.
